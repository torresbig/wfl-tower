## 💡 Schnell-Tipps

**Texte anpassen?**  
→ Öffne `src/app/components/StorySection.tsx`

**Mehr Bilder hinzufügen?**  
→ Hochladen in `public/bilder/slide/` und Namen in `ImageSlideshow.tsx` eintragen

**Farben ändern?**  
→ Standard ist Rot (Wings for Life) - änderbar in den Component-Dateien

**Lokal testen vor Deployment?**  
→ `pnpm install` dann `pnpm dev`

## 🆘 Hilfe benötigt?

1. Schau in [DEPLOYMENT.md](DEPLOYMENT.md) unter "Häufige Probleme"
2. Überprüfe die GitHub Actions Logs (Tab "Actions" in deinem Repo)
3. Stelle sicher dass die `base` URL in `vite.config.ts` korrekt ist
